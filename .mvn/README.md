# Selenium Test Project

## 📌 Description
Ce projet consiste à réaliser des tests automatisés en utilisant Selenium avec Java.

## 🛠️ Technologies utilisées
- Java
- Selenium WebDriver
- TestNG / JUnit
- Maven
- IntelliJ IDEA

## 🧪 Test réalisé
Le test permet de :
- Lancer un navigateur (Chrome)
- Accéder à un site web
- Vérifier le titre de la page

## ▶️ Exécution
1. Lancer la classe de test
2. Le navigateur s’ouvre automatiquement
3. Le test s’exécute
